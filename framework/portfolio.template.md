## Anti-vanity check (monthly)

When updating this ledger, ask:

1. **Could a third party actually find these?** (Linked from a public profile? Findable via search?)
2. **Could you demo them in 5 minutes if asked?** (For builds: still running? Recoverable?)
3. **Could you answer a hostile follow-up?** (For writeups: do you remember the decisions you made? For case studies: have you rehearsed the difficult questions?)

If any answer is no, fix it before adding more items.
